Guía Básica de HTML, Hosting y GitHub

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje principal para crear sitios web. Permite estructurar páginas usando etiquetas para insertar textos, imágenes, enlaces, videos y mucho más.

Gracias a HTML, los navegadores como Chrome o Firefox pueden mostrar contenido ordenado, entendible y visualmente atractivo para los usuarios. Es la base de toda página web junto con CSS (diseño) y JavaScript (funcionalidad).

HTML no es un lenguaje de programación, ¡es un lenguaje de marcado!

¿Qué es el Hosting?

El hosting es un servicio que permite guardar tu sitio web en un servidor para que esté disponible en internet 24/7. Existen opciones gratuitas y de pago, según lo que necesites.

Algunos proveedores conocidos son: Hostinger, Bluehost, GoDaddy y GitHub Pages (ideal para proyectos pequeños).

¿Qué es GitHub?

GitHub es una plataforma donde puedes guardar tu código y compartirlo con el mundo. Es muy usada por desarrolladores para trabajar en equipo, controlar versiones de sus proyectos y publicar sitios web sencillos.

Puedes crear una cuenta gratuita en github.com y comenzar a subir tus proyectos web.
